# Reto tecnico - Franco Sotelo

## 1. Mi experiencia usando ASP.NET 

En el dia a dia en mi trabajo actual uso este framework para realizar nuevos features o mantenimiento, 

los clientes principales donde uso este framework es en Luz del Sur y Sanna.

### Backend (Servidor)

- Desarrollado en ASP.NET Core 9.0,

use el 9.0 (ultima version) ya que estoy en un ambiente libre(local) y no tengo limitantes impuestos por clientes que suelen

usar versionas mas antiguas

- Expuse un endpoint REST para el consumo por parte del cliente(index.html)

### Frontend (Cliente)

- Aplicación Single Page Application (SPA).

- Construida con HTML5, CSS3 y JavaScript.

- Consume de forma asíncrona la API local mediante peticiones HTTP.

## 2. Especificaciones Técnicas

- **Consumo de API Externa:**  

  Uso de IHttpClientFactory para realizar peticiones seguras y eficientes a la API brindada: https://randomuser.me.

- **Mapeo de Datos:**  

  Implementación de la clase Persona en la capa Models, garantizando tipado fuerte y una estructura clara de la información.

- **Interfaz de Usuario:**  

  - Diseño responsive usando CSS Grid.  

  - Renderizado dinámico de los datos mediante JavaScript.

## 3. Instrucciones para ejecutar el proyecto

1. Abrir la solucon del proyecto en Visual Studio.

2. Compilar y ejecutar el proyecto usando  https

3. Ir a la raíz del sitio web para visualizar
